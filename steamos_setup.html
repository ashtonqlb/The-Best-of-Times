Installation

Preface

A clean installation is not only required to get rid of any potential leftover mod files, but also to make sure the game is installed in a safe location. Here is what you need to know when choosing where to install your game:
  • Install on internal storage if possible: Having the game on your device's internal SSD will drastically improve loading times and decrease stuttering.
  • Fallout 3 Installation: Fallout 3 only needs to be installed for running the TTW installer, and can be uninstalled after the installer completes.

Uninstalling the Games

If you have never installed the games before, go directly to the next section.

As the games are available on Steam, GOG, and Epic Games, there are separate instructions for each version. You obviously only need to follow the directions for the versions you have.

Steam

If you already uninstalled the games, make sure to follow step 4 to ensure there are no leftovers.

Fallout: New Vegas

  1. Open Steam and go to your Library.
  2. Find Fallout: New Vegas in the list.
  3. Right-click on it and select Manage -> Uninstall.
  4. Navigate to ~/.local/share/Steam/steamapps/compatdata/ and delete the folder named 22380

Fallout 3

  1. Open Steam and go to your Library.
  2. Find Fallout 3 in the list.
  3. Right-click on it and select Manage -> Uninstall.
  4. Navigate to ~/.local/share/Steam/steamapps/compatdata/ and delete the folder named 22370

Fallout: New Vegas

  1. Open Heroic Games Launcher and go to your Library.
  2. Find Fallout: New Vegas in the list.
  3. Right-click on it and select Uninstall.
    • At the prompt, Tick both Remove prefix and Erase settings & remove log

Fallout 3

  1. Open Heroic Games Launcher and go to your Library.
  2. Find Fallout 3 in the list.
  3. Right-click on it and select Uninstall.
    • At the prompt, Tick both Remove prefix and Erase settings & remove log

Installing the Games

Although Fallout: New Vegas is available on multiple storefronts, the installation process is almost exactly the same. Note that the GOG & Epic versions of the game must be installed with Heroic Games Launcher. Steam versions, obviously, can be installed with Steam.

You do not need to install multiple copies of the game if you have it on multiple platforms

Steam

Fallout: New Vegas

  1. Open Steam and go to your Library.
  2. Find Fallout: New Vegas in the list and select Install.
  3. Set the Install to location to a location of your choice.

Fallout 3

  1. Open Steam and go to your Library.
  2. Find Fallout 3 in the list and select Install.
  3. Set the Install to location to a location of your choice.

Post-Installation

Key Terminology

Now that the game is installed, there are three folders from it that will be referred to in the guide often:
  • Root folder: Where the game is installed.
    • For Steam:

      ~/.local/share/Steam/steamapps/common/Fallout New Vegas/

  • Data folder: Where all of the game's assets are located.
    • For Steam:

      ~/.local/share/Steam/steamapps/common/Fallout New Vegas/Data

  • Prefix folder: Where all of the game's configuration & Windows compatibility files are located.
    • For Steam:

      ~/.local/share/Steam/steamapps/compatdata/22380/pfx

Installing Components

Proton, the compatibilty layer used to run Fallout: New Vegas on SteamOS 3, lacks some important additional Windows libraries that we must install manually.

Steam

Installing Visual C++

  1. Enter Desktop mode.
  2. Install Protontricks from the Discover store.
  3. Open Protontricks and select Fallout New Vegas -> OK.
  4. Select Select the default wineprefix -> OK.
  5. Select Install a Windows DLL or component -> OK.
  6. Scroll down to and check the box for vcrun2022 then select OK.
  7. Proceed with installing Visual C++ as prompted. If prompted, feel free to ignore any SHA-1 hash warnings.
  8. Once installation has been completed, exit the program by clicking Cancel.

Generating Fresh INI Files

Fallout: New Vegas

  1. Run the game from Steam/Heroic Games Launcher
  2. Click OK to both pop-ups that say Detecting Video Hardware.
    • If there aren't any pop-ups, delete your Prefix folder, then retry.
  3. Click OK then Exit.

Fallout 3

  1. Run the game from Steam/Heroic Games Launcher
  2. Click OK to both pop-ups that say Detecting Video Hardware.
    • If there aren't any pop-ups, delete your Prefix folder, then retry.
  3. Exit the launcher.
