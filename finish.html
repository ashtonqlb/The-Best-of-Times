Manual Downloads

Root Mods

The following mods, unlike most others, need to be installed directly into the game's Root folder, not through Mod Organizer 2

Use the Manual Download and follow the instructions provided for each mod.

Game Patchers

Each version of the game requires a different patcher to work with xNVSE.
These patchers allow the game to use up to 4GB of RAM and automatically load xNVSE.

Select one of the following patchers based on your game version.

Steam and GOG

4GB Patcher

Main Files - FNV4GB for Linux
  1. Open the downloaded archive and move the executable file to the game's Root folder.
  2. Double-click on FalloutNVPatcher to run it.
  3. A Konsole window will open and it should say:
    FalloutNV.exe patched!
  4. Close Konsole and a file named FalloutNV_backup.exe should appear in the game's Root folder.

Epic Games

Epic Games Patcher

Main Files - Epic Games Patcher
  1. Extract the downloaded archive to your Downloads folder.
  2. Download Delta Patcher from the Discover Store.
  3. Open Delta Patcher
  4. In the box marked Original File click the folder icon and use the resultant file picker menu to select FalloutNV.exe in the game's Root folder.
  5. In the box marked XDelta Patch click the folder icon and use the resultant file picker menu to select patch.xdelta from the Epic Games Patcher folder.
  6. Click Apply patch.
  7. Once the game has finished patching you may close the window.

Setup Finish

Installing The Best of Times

You have now finished the missing SteamOS 3 setup section of The Best of Times. You can proceed with the rest of the guide, but be aware that you only need to follow the Performance Mods section of the Performance and Stability Guide. You may optionally follow the guidance of the FPS Limits section

Why skip most of the Performance and Stability guide?

Most (not quite all) of that section of the guide is concerned with making Fallout use a more modern rendering pipeline. SteamOS already ships with a modern rendering pipeline, and it's built in Game Mode session comes with pretty much everything you would have to do by hand (Including the frame limiter!).

That all said, performance mods are performance mods and will improve your framerate/game stability. So that bit, you should absolutely pay attention to.
