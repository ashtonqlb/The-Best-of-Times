Pre-Installation

Preparing Hoolamike

This utility is currently under active development, and as such these instructions may be subject to change at any time. Due to the rapidly-evolving nature of the software, Absolutely no support can be offered at this time.

This utility has no GUI and commands must be either pasted or typed manually. While every effort has been made to make this process as painless as possible without a dedicated keyboard and mouse, you will likely still need to perform some manual keyboard input. The virtual keyboard can be invoked at any time in desktop mode with Steam + X as long as Steam is running

Hoolamike is a free and open source replacement for the Tale of Two Wastelands installer for macOS and Linux, written in Rust. Before installing Hoolamike, make sure that you have installed the latest stable version of Rust.

Once you have downloaded and extracted Hoolamike to a folder of your choosing, you must configure it like so:
  1. Open your extracted Hoolamike folder in Dolphin. Right click "Open in terminal"
  2. Paste chmod u+x ./hoolamike into the Terminal. This will allow you to execute the program.
  3. Paste ./hoolamike print-default-config > hoolamike.yaml into the Terminal window. This will create a default configuration file.
  4. You will need to add some extra variables to the generated hoolamike.yaml file, they are as follows
    • For Steam, you will need to paste
      games:
       FalloutNewVegas:
        root_directory: ~/.local/share/Steam/steamapps/common/Fallout New Vegas/
       Fallout3:
        root_directory: ~/.local/share/Steam/steamapps/common/Fallout 3 goty/
      fixup:
       game_resolution: 1280x800 # If your device supports a different resolution, make sure to change it.
      extras:
       tale_of_two_wastelands:
         path_to_ttw_mpi_file: # Replace this with the path to the Tale of Two Wastelands .mpi file
         variables:
          USERPROFILE: ~/.local/share/Steam/steamapps/compatdata/22380/pfx/drive_c/users/steamuser/Documents/My\ Games/FalloutNV/
          DESTINATION: ~/Tale\ of\ Two\ Wastelands/

      Into your hoolamike.yaml.
    • For Epic Games & GOG, you will need to paste
      games:
       FalloutNewVegas:
        root_directory: ~/Games/FalloutNewVegas/ # Change this if you did not install in the default directory
       Fallout3:
        root_directory: ~/Games/Fallout3 # Change this if you did not install in the default directory
      fixup:
       game_resolution: 1280x800 # If your device supports a different resolution, make sure to change it.
      extras:
       tale_of_two_wastelands:
         path_to_ttw_mpi_file: # Replace this with the path to the Tale of Two Wastelands .mpi file
         variables:
          USERPROFILE:~/Games/Heroic/Prefixes/default/Fallout New Vegas/pfx/drive_c/users/steamuser/Documents/My\ Games/FalloutNV/
          DESTINATION: ~/Tale\ of\ Two\ Wastelands/

      Into your hoolamike.yaml.
  5. Once you have pasted in the additional variables, save your changes and exit.

Installation

Installing Tale of Two Wastelands

  1. Download the Tale of Two Wastelands Installer and extract the contents of the archive anywhere.
  2. Create a folder named Tale of Two Wastelands and then copy its path.
  3. Open your hoolamike.yaml and verify that the set file paths for both your Fallout 3 and Fallout: New Vegas installations are correct.
  4. Copy the path of the TTW installer .mpi file to your clipboard and paste it into the field path_to_ttw_mpi_file:
  5. Ensure your paths look similar to the example above, then open a terminal in your Hoolamike directory
  6. Paste hoolamike tale-of-two-wastelands into the terminal
  7. The final folder's size should be ~17GB, reinstall Tale of Two Wastelands if it isn't. This can be done by first clearing the contents of the DESTINATION folder defined in your hoolamike.yaml, then repeating the installation steps.

The installation will take a long time as the audio re-encoding section is very CPU-intensive. Installation on a Steam Deck takes approximately two hours.

It is recommended to close any other CPU-intensive application during this time to avoid slowdowns.

If you run into any errors during the installation, check the solutions listed in the FAQ.

Importing and Deploying

  1. Open Limo.
  2. Drag your Tale of Two Wastelands folder into the main pane. If you installed TTW correctly, you should see an installer pop up. Importing the mod may take up to five minutes.
  3. When the installer pops up, proceed with the default settings.
  4. Go to the Deployers tab.
  5. Select the Data deployer and click Deploy.
  6. Right click anywhere in the Overview and then click Sort Mods
