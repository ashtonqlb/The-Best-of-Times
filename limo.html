Limo

What is Limo?

Limo is an open source game mod manager, mainly used for Bethesda games. Its main features are:
  • Union Filesystem (Staging Directory): Limo does not install mods directly into the game's folder, but rather creates a virtual filesystem that the game reads from. This allows for:
    • Safe mod installation and uninstallation,
    • Being able to modify mod install order at any point,
    • Having multiple profiles with different mod setups,
    • Keeping original game files clean.
    The downsides to having a staging directory are:
    • Mods must be re-deployed after each change to the load order.
    • Some programs may not work properly with Limo's virtual filesystem,
    • It can cause performance overhead if you have tons of loose (not-BSA/BA2) files,
  • Profile System: Limo allows you to create multiple profiles with different mod setups, saves, and settings.
  • Tabbed interface: Limo is split into two sides, the left side is the mod list and the right side is the plugin list. This allows for easy conflict resolution and mod management.
  • Modding Community Integration: Limo has built-in support for Nexus Mods, allowing you to download mods directly from the Nexus.

    Limo is currently in active development and as such, major functionality may be subject to change at any time. Because of this, be aware that No support whatsoever can be offered at this time.

    Limo has NO support for Nexus' Collections feature, so you will have to download mods one by one.
    This is not a big issue, as the guide will provide direct links to the mods you need, and most collections are low quality and not curated.

    The closest alternative to Collections is Wabbajack, which is a community-made modlist installer that can install entire modlists with a single click. Due to its higher bar of entry for modlist creators, Wabbajack lists tend to be of higher quality compared to Collections.

Why Limo?

Limo has been chosen for this guide due to its ease of use and its ability to keep the game's folder clean. This is important because it allows for easy conflict resolution and mod management, and makes correcting mistakes trivial - installed mods in the wrong order? Just drag them into the correct position!

Compared to Vortex, Limo doesn't hide the most important parts of modding from the user, making it easier to understand what is happening and why, albeit at the cost of potentially looking scary at first.

That said, we are well aware that Limo is not perfect, and it has its own set of issues and limitations, but it is the best tool for the job.

Installation

Installing Limo

    If you are comfortable using the terminal, you can skip this section with the command flatpak install io.github.limo_app.limo

  1. Download Limo from here.
  2. Once the download has finished, click on the downloaded file.
  3. Click install.

Setup

Configuring Limo

On first launch you will be greeted with a dialog for adding a new target application to Limo. Currently, Limo supports importing installed games from Steam, but not from Heroic (GOG & Epic Games.) As such, separate instructions have been provided.

Steam

  1. Select Import from Steam
  2. Select Fallout New Vegas
  3. Select a staging directory of your choosing. This will be where mod management and downloads occur
  4. Navigate to the Deployers tab

Configuring Settings

  1. Click the button at the top right corner of Limo to open the settings.
  2. In the NexusMods tab, select Set API key.
  3. Open this link to retrieve your Mod Organizer 2 API key. This key will also work with Limo.
  4. Select the Copy icon to retrieve your MO2 API key
  5. Returning to Limo, select Set API Key, then paste your key into the field
  6. Optionally, you can set a custom decryption password. This can help prevent your credentials from being stolen.

Game Settings

  1. Open the game launcher by pressing the Launch Application option in the top-left corner
  2. Inside the launcher, click Options and do the following:
    1. Select the High Preset.

      If you want longer battery life, you can select the Medium Preset instead.

  3. Close the launcher.

Mod Installation Instructions

How to use Nexus and Limo?

When the guide says to install a mod, it will be formatted as Main File - Example 1.5, this is referring to the mod's category on Nexus and file name.

IMPORTANT:
Unless otherwise specified, always use Mod Manager Download.



Standard procedure through the manager:
  1. Download the indicated file(s).
  2. Wait for the Install Mod dialogue to pop up
  3. Click OK to install it.

If downloading through the manager is not an option, do the following:
  1. Click the Manual Download button (or whatever the download button is for the respective site).
  2. Once the download has finished, drag the downloaded file into Limo
  3. Wait for the Install Mod dialogue to pop up. This may take a while for larger mods.
  4. Finish the install as you would with a Mod Manager Download mod.

Add to deployer will appear in the bottom right corner of every mod installation. You must make sure that you are always adding mods to the Data deployer. This can be done by checking the box marked Data if it is not already.


When installing a mod, Limo might prompt you with multiple choices:
  • Rename appears when you are trying to install a file with the same name as an existing one. If you are installing, for example, a secondary file necessary for the mod to work, then rename it accordingly with the name drop-down list or by typing the name manually.
  • If you are updating from an old version of a mod, you should select Replace, this will automatically delete all the files from the old version of the mod and replace them with the ones from the new version.
  • Merge should only be used when you are sure the mod will not be updated and in situations where a hotfix is uploaded as separate file. Avoid it if you can, for maximum modularity.
